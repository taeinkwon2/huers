Brand Concept
Brand Concept
HUERS는 개인의 취향을 하나의 고정된 선택이 아닌, 여러 감각이 겹쳐 만들어지는 블렌드로 해석하는 커피 브랜드입니다.
각기 다른 취향의 레이어가 중첩되며 새로운 커피 경험을 만들어냅니다. 개인화, 블렌딩, 정제, 투명성, 그리고 감각적 깊이를 핵심 가치로 삼습니다.

2.1 Symbol
2.1 Symbol
HUERS의 심볼은 단일한 형태가 아니라 유기적인 투명 레이어의 조합으로 구성됩니다. 각 레이어는 서로 다른 취향 요소를 의미하며, 중첩되는 지점에서 새로운 색과 깊이가 생성됩니다.

심볼은 워드마크 없이 독립적으로 사용될 수 있으나, 브랜드 인지도가 충분히 확립된 공식 채널에서만 단독 사용을 허용합니다.

2.2 Wordmark
HUERS 워드마크는 세리프 기반의 타이포그래피를 통해 정제된 감성, 안정감, 프리미엄 이미지를 전달합니다.
변화하는 그래픽 요소 속에서도 워드마크는 브랜드의 중심축 역할을 합니다.

워드마크는 항상 원본 파일을 기준으로 사용하며, 임의로 재작도하거나 변형해서는 안 됩니다.

Wordmark Principle
Wordmark Principle
워드마크는 항상 명확한 가독성을 유지해야 하며, 레이어 그래픽보다 상위에 배치됩니다. 그래픽 요소는 워드마크를 보조해야 하며, 시각적 중심을 방해해서는 안 됩니다.
워드마크의 레이어 우선순위는 항상 최상위를 유지합니다. 배경 → 투명 레이어 그래픽 → HUERS 워드마크 순서로 구성하며, 이 원칙은 모든 적용물에서 동일하게 적용됩니다.

2.4 Color System
2.4 Color System
HUERS는 하나의 고정된 브랜드 컬러를 정의하지 않습니다. 브랜드 컬러는 반투명 레이어의 중첩과 빛의 흐름을 통해 유동적으로 생성됩니다.
색은 고정된 정체성이 아니라, 블렌딩의 결과입니다. HUERS의 컬러는 항상 관계 속에서 의미를 가집니다.

2.6 Incorrect Usage
2.6 Incorrect Usage
HUERS 로고는 브랜드의 핵심 자산이므로 임의 변형을 금지합니다. 아래의 사용 방식은 모두 브랜드 가이드라인 위반에 해당합니다.
❌ 로고 왜곡 금지
Distort the logo — 가로 또는 세로 방향으로 늘이거나 압축하지 않습니다.
❌ 회전 금지
Rotate the logo — 로고를 어떠한 각도로도 회전시키지 않습니다.
❌ 비율 변경 금지
Change the wordmark proportion — 워드마크의 자간, 크기 비율을 임의로 변경하지 않습니다.
❌ 레이어 임의 제거 금지
Remove essential layers — 심볼을 구성하는 레이어를 임의로 삭제하지 않습니다.
❌ 외곽선 추가 금지
Add random outlines — 워드마크나 심볼에 임의의 외곽선을 추가하지 않습니다.
❌ 저대비 배경 금지
Use low-contrast backgrounds — 가독성을 해치는 배경 위에 로고를 배치하지 않습니다.
❌ 강렬한 색상 적용 금지
Apply harsh colors — 브랜드 무드를 훼손하는 고채도·원색 컬러를 적용하지 않습니다.

2.7 Brand Definition
